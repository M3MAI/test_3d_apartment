# مصمم الشقة التفاعلي — Apartment Designer

موقع ثابت (HTML / CSS / JavaScript بدون أي بناء) لاستعراض تخطيط شقة ثلاثية الأبعاد تم استخراجه من فيديو، ولتجربة ترتيب العفش في كل غرفة بشكل تفاعلي (drag & drop).

## الاستخدام السريع

```bash
# افتح الموقع مباشرة
open index.html
# أو قدم الملفات عبر خادم محلي
python3 -m http.server 8000
# ثم افتح: http://localhost:8000
```

## المحتوى

- `index.html` — الصفحة الرئيسية.
- `css/styles.css` — التنسيقات.
- `js/rooms.js` — تعريف الغرف (الأبعاد، الأبواب، الشبابيك).
- `js/furniture.js` — كتالوج العفش (الاسم، الأبعاد، الأيقونة).
- `js/app.js` — منطق التطبيق (السحب والإفلات، الحفظ، التصدير).
- `docs/ANALYSIS.md` — تقرير تحليل الشقة المأخوذ من الفيديو + صور مرجعية.
- `docs/frames/` — لقطات مفتاحية من الفيديو.

## الميزات

- **اختيار الغرفة**: قائمة جانبية بجميع غرف الشقة (صالون، صالة، 3 غرف نوم، مطبخ، حمامان).
- **إسقاط العفش**: اسحب أي قطعة من كتالوج العفش وأسقطها داخل الغرفة.
- **تحريك / دوران / تكرار / حذف**: اضغط على قطعة ثم استخدم اللوحة اليسرى أو اختصارات لوحة المفاتيح `R` للدوران، `D` للتكرار، `Del` للحذف.
- **شبكة ثبات Grid Snap**: كل القطع تنجذب لأقرب 5 سم.
- **حفظ تلقائي في المتصفح**: الترتيب يُحفظ في `localStorage`.
- **تصدير / استيراد JSON**: يمكنك تصدير تصميمك ومشاركته مع شخص آخر أو العودة إليه لاحقًا.
- **دعم اللمس والفأرة** على الموبايل والديسكتوب.

## التحليل المعماري

راجع [`docs/ANALYSIS.md`](docs/ANALYSIS.md) لجدول الغرف والأبعاد التقريبية وصور مأخوذة من الفيديو.

## ⚠️ تنبيه مهم بشأن الأبعاد

الأبعاد في `js/rooms.js` **مُستنتَجة بصرياً من الفيديو** ولم تُقَس على الطبيعة. الأرقام مقرّبة (500، 400، 350 سم) وقد تختلف عن الأبعاد الفعلية بـ ±20 سم لكل غرفة. كذلك ألوان الجدران مأخوذة من عيّنات بصرية بدقة ±10 لكل قناة RGB.

**قبل استخدام التطبيق لطلب أثاث حقيقي أو تنفيذ تشطيبات**: قِس الغرف بمتر القياس وحدّث `js/rooms.js` (أو استخدم محرر الغرفة في التطبيق لتعديل العرض/العمق/الارتفاع) أو استورد من ملف DXF من AutoCAD.

أيضاً **الممر المركزي بين الغرف غير مُمثَّل حالياً كغرفة مستقلة** — الغرف موضوعة على إحداثيات تقريبية في خطة ثنائية الأبعاد. هذا يكفي لتجربة الأثاث داخل كل غرفة لكن لا يمثّل مسار المشي الفعلي بين الغرف بدقة.
